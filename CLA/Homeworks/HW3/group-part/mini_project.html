From the graph, we can conclude the following four equations. \[0 = -gm_{1} + k(L-(y_{1}-0)) - k(L-(y_{2}-y_{1}))\] \[0 = -gm_{2} + k(L-(y_{2}-y_{1})) - k(L-(y_{3}-y_{2}))\] \[0 = -gm_{3} + k(L-(y_{3}-y_{2})) - k(L-(y_{4}-y_{3}))\] \[0 = -gm_{4} + k(L-(y_{4}-y_{3})) - k(L-(H-y_{4}))\] And we can change the equations into the matrix form.

\[\begin{pmatrix} -2k&k&0&0\\k&-2k&k&0 \\0&k&-2k&k \\ 0&0&k&-2k\end{pmatrix} \begin{pmatrix}y_{1}\\y_{2}\\y_{3}\\y_{4}\end{pmatrix} = \begin{pmatrix}gm_{1}\\gm_{2}\\gm_{3}\\gm_{4}-kH\end{pmatrix}\]

Now we can pick some parameters and solve the matrix. We pick \(k = 5N/kg\), \(g = 10 m/s^{2}\), \(m_1 = m_2 = m_3 = m_4= 1kg\), \(H = 40m\).

# initialize variables 
k <- 5
g <- 10
m1 <- 1
m2 <- 1
m3 <- 1
m4 <- 1
H <- 40
A <- cbind(c(-2*k, k, 0, 0), c(k, -2*k, k, 0), c(0, k, -2*k, k), c(0, 0, k, -2*k))
b <- c(g*m1, g*m2, g*m3, g*m4-k*H)
A
##      [,1] [,2] [,3] [,4]
## [1,]  -10    5    0    0
## [2,]    5  -10    5    0
## [3,]    0    5  -10    5
## [4,]    0    0    5  -10
b
## [1]   10   10   10 -190
x <- qr.solve(A,b)
x
## [1]  4 10 18 28

Here, we get the result x as \(\begin{pmatrix}4 \\ 10 \\ 18 \\ 28\end{pmatrix}\)

Just as shown in the graph above, the four objects are at the hegiht of 4m, 10m, 18m, 28m. And the total height is 40m.
